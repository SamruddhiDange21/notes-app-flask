# Notes App (Flask)

## Description
A simple full-stack notes application built using Python Flask, HTML, CSS, and JavaScript.  
Users can add notes, view notes, and delete notes. Notes are stored in a JSON file.

## Features
- Add notes
- Delete notes
- View saved notes
- Data stored using JSON

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run

1. Install Flask
```
pip install flask
```

2. Run the application
```
python app.py
```

3. Open the browser
```
http://127.0.0.1:5000
```
